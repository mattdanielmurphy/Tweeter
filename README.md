# Tweeter

Tweeter is a simple, single-page Twitter clone.

Built with HTML, CSS, JS, jQuery, AJAX, NodeExpress and MongoDB.

## Dependencies

- Express
- Node 5.10.x or above
- Body Parser
- Chance
- MD5
- MongoDB
- TimeAgo
